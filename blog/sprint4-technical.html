Solved a Problem in an Elegant Way

I found that I was able to get the Super FizzBuzz array working much quicker than I anticipated. I called back to the task where I used the remainder operator “%”, to return fizz, buzz or fizzbuzz to their appropriate numbers.

So, I wrote the function that would take in the number in the array, checks if they divide cleanly by 3, 5 then 15 and return either fizz, buzz, fizzbuzz or just the number itself if there was a remainder.

I used a lot of pseudocode when going through the problems on repl.it.
